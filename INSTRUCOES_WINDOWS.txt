    PHOTO MOSAIC - INSTRUÇÕES WINDOWS
========================================

PROBLEMA COMUM: "python não é reconhecido como comando"

SOLUÇÃO:

1. INSTALAR PYTHON:
   - Baixe Python em: https://www.python.org/downloads/
   - Durante a instalação, MARQUE "Add Python to PATH"
   - Reinicie o computador após a instalação

2. TESTAR PYTHON:
   - Abra o Prompt de Comando (cmd)
   - Digite: python --version
   - Se mostrar a versão, está funcionando

3. EXECUTAR A APLICAÇÃO:
   - Duplo clique em: teste_python.bat
   - Se funcionar, use: iniciar_windows.bat

4. SE AINDA NÃO FUNCIONAR:
   - Abra o Prompt de Comando como Administrador
   - Navegue até a pasta do projeto
   - Execute: python server_windows.py

5. ALTERNATIVA:
   - Instale Python 3.11 ou superior
   - Certifique-se de marcar "Add Python to PATH"
   - Reinicie o computador

ARQUIVOS IMPORTANTES:
- teste_python.bat (testa se Python funciona)
- iniciar_windows.bat (inicia a aplicação)
- server_windows.py (servidor compatível com Windows)

CONTATO:
Se ainda não funcionar, verifique se Python está instalado corretamente.
